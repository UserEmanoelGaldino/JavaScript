    Node.js:

    var n = 5
    typeof n

    Cadeia de caracteres - strings
    Valor Booleano (True or False) - Boolean

FORMAS DE DECLARAR STRINGS:
    Em JS, para informar uma string pode-se usar três tipos de aspas, que são:
    '' - Aspas Simples
    "" - Aspas Duplas
    `` - Craze

NOME DE VÁRIÁVEL SÃO IDENTIFICADORES [REGRAS]:

    - Podem começar com: letra, $ ou _
    - Não podem começar com número;
    - É possível usar letras e números;
    - É possível usar acentos e símbulos;
    - Não podem conter espaços;
    - Não podem ser palavras reservadas;

DICAS PARA ESCOLHER NOME DE IDENTIFICADOR:
    - Maiúsculas e minúsculas fazem diferença; (Sensitive Case)
    - Utilizar nomes coerentes para os valores que a variável irá receber;
